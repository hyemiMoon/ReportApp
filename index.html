구글 스프레드시트 연동 설정

* 시트 연동을 위한 전용 앱스 스크립트 배포 주소를 입력하세요.

1 구글 스프레드시트 준비

  1. 사용하실 스프레드시트를 열고 시트 이름을 시트1(기본값)로 둡니다.
  2. 첫 줄에 날짜, 신고 유형, 내용 헤더를 만듭니다 (필수는 아니지만 구분을 위함).

2 앱스 스크립트 작성 및 배포

  1. 스프레드시트 상단 메뉴의 [확장 프로그램] > [Apps Script]를 클릭합니다.
  2. 기존 코드를 지우고, 아래의 소스코드를 그대로 복사하여 붙여넣기 합니다.
function doPost(e) {
  try {
    // 본인의 구글 스프레드시트 ID를 입력하세요.
    // (예: https://docs.google.com/spreadsheets/d/시트_아이디/edit#gid=0)
    var spreadsheetId = "1vFvWuYjz5qvZiLNlMfEhZaP4cHeT3vrcQU5Vko_tyrY"; 
    var sheet = SpreadsheetApp.openById(spreadsheetId).getSheets()[0];
    
    var parameter = e.parameter;
    
    // JSON 요청 지원 (API 호환성 향상)
    if (e.postData && e.postData.contents) {
      parameter = JSON.parse(e.postData.contents);
    }
    
    // 대한민국 서울 시간으로 포맷 설정
    var date = Utilities.formatDate(new Date(), "Asia/Seoul", "yyyy-MM-dd HH:mm:ss");
    var type = parameter.type || "기타";
    var content = parameter.content || "내용 없음";
    
    // 스프레드시트 끝 줄에 날짜, 신고 유형, 내용 추가
    sheet.appendRow([date, type, content]);
    
    return ContentService.createTextOutput(JSON.stringify({"result": "success", "date": date}))
                         .setMimeType(ContentService.MimeType.JSON)
                         .setHeader("Access-Control-Allow-Origin", "*");
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({"result": "error", "message": error.toString()}))
                         .setMimeType(ContentService.MimeType.JSON)
                         .setHeader("Access-Control-Allow-Origin", "*");
  }
}
  1. 상단 💾(저장) 버튼을 클릭합니다.
  2. 우측 상단 [배포] > [새 배포]를 누릅니다.
  3. 유형 선택에서 [웹 앱]을 지정합니다.
  4. 설명은 자유롭게 적고, 액세스 권한이 있는 사용자를 반드시 [모든 사용자] (Anyone)로 설정한 후 배포합니다. (매우 중요!)
  5. 처음 배포 시 권한 검토 요구가 나옵니다. [권한 검토] > 구글 계정 선택 > [Advanced] > [Go to 무제한 프로젝트(unsafe)] > [Allow]를 차례대로 승인합니다.
  6. 생성된 웹 앱 URL 주소를 복사하여 위의 설정 입력 칸에 붙여넣고 저장하세요!
